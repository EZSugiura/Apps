# プライバシーポリシー

本アプリ「ファブルからの挑戦状」（以下「本アプリ」）は、ユーザーの皆様に安心してご利用いただけるよう、以下のとおりプライバシーポリシーを定めます。

## 1. 収集する情報

本アプリは、ユーザーを個人として特定する情報（氏名、メールアドレス、電話番号等）を直接収集することはありません。

本アプリでは、以下の情報が取得・利用されることがあります。

### 広告配信（Google AdMob）

本アプリでは、広告配信のために Google AdMob を利用しています。広告配信にあたり、Google および広告配信パートナーが広告識別子（Advertising ID）等の情報を Cookie やこれに類する技術を用いて取得し、利用することがあります。

これらの情報の取り扱いについては、Google のプライバシーポリシーをご確認ください。
- Google プライバシーポリシー: https://policies.google.com/privacy
- Google が広告に使用するデータについて: https://policies.google.com/technologies/ads

### 統計情報（Firebase Cloud Firestore）

本アプリでは、ゲームの利用状況に関する統計（挑戦者数、各問題の突破者数など）を Firebase Cloud Firestore に記録しています。これらはユーザー個人を特定しない集計データのみであり、個人情報は含まれません。

### 端末センサー（加速度センサー）

本アプリの一部のギミックでは、端末を振る操作を検出するために加速度センサーの値を利用します。これらの値は端末内でのみ処理され、外部に送信・保存されることはありません。

### ローカル保存データ

ゲームの進行状況などは、端末内（shared_preferences）にのみ保存され、外部サーバーへ送信されることはありません。

## 2. 情報の利用目的

取得した情報は、以下の目的にのみ利用します。

- 広告の配信および表示
- アプリの利用状況の把握、品質改善
- ゲームの進行状況の保存・復元

## 3. 第三者への提供

本アプリは、上記「収集する情報」に記載した範囲を除き、取得した情報を第三者に提供することはありません。

## 4. 免責事項

本アプリの利用により発生した、いかなる損害についても、開発者は責任を負いません。

## 5. プライバシーポリシーの変更

本ポリシーの内容は、必要に応じて事前の予告なく変更することがあります。変更後のプライバシーポリシーは、本ページに掲載した時点で効力を生じるものとします。
